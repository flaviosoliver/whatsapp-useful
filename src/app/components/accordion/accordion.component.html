 Informações de uso, clique para {{ panelOpenState ? "recolher" : "expandir" }}
  • Escolha de Plataforma:
    • WhatsApp Web: abrir o WhatsApp diretamente no navegador web.
    • App WhatsApp: irá executar a aplicação do WhatsApp existente em seu sistema operacional, caso não seja localizado, será executado no navegador.
  • Telefone
    • Informe o número de telefone ao qual deseja entrar em contato pelo WhatsApp. Insira um número de telefone em formato válido.
  • Com mensagem
    • Nessa caixa de seleção, você controla se deseja que seu primeiro contato com esse número aconteça já com uma mensagem pré-carregada.
  • Caixa de mensagem
    • Observando o controle de seleção acima, através dessa caixa de texto, você poderá adicionar a mensagem inicial dessa interação via WhatsApp. Quando abrir a tela do aplicativo a mensagem já estará lá pronta para ser enviada.
  • Botão Enviar
    • Como o nome do botão já propõe, através dele irá abrir uma nova janela com a API oficial do WhatsApp, caso não esteja logado no WhatsApp, a API retornará a tela de login oficial exibindo o QR Code para tal conexão.
  • Botão Gerar QR Code
    • Com esse botão é possível gerar um QR Code com os dados passados nos itens anteriores (Telefone e/ou Mensagem). Uma aplicabilidade real nisso é poder submeter essa conversa para um outro aparelho, desfazendo a necessidade de abrir conexão nesse que está em uso o site.
