info Sobre o Projeto

Motivação

Este projeto veio com dupla motivação: resolver uma dor sobre ter que conversar com alguém no WhatsApp sem ter o interesse de salvar seu contato e seguir estudando e desenvolvendo habilidades no framework Angular.

Desafio

Criar uma página Web de layout agradável onde seja possível chamar a API do WhatsApp e poder iniciar uma conversa sem a premissa de ter esse contato salvo em minha agenda.

Este site não coleta ou armazena nenhuma informação pessoal ou dados utilizados no preenchimento dos seus campos de formulário.
