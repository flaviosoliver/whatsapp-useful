rocket_launch Sobre esse Projeto

Motivação:

Esse projeto veio com dupla motivação, resolver uma dor sobre ter que conversar com alguém no WhatsApp sem ter o interesse de salvar seu contato e seguir estudando e desenvolvendo habilidades no framework Angular.

Desafio:

Criar uma página Web de layout agradável utilizando Angular v15 onde seja possível chamar a API do WhatsApp e poder iniciar uma conversa sem a premissa que de ter esse contato salvo em minha agenda.

Recursos Utilizados:

  • TypeScript
  • SASS
  • Angular v15
  • Lint
  • Angular Forms
  • Ngx Mat Intl Tel Input
  • Angular Material
  • Material Icons
  • Customização do Material

Infos:

  • Escolha de Plataforma:
    • WhatsApp Web: abrir o WhatsApp diretamente no navegador web.
    • App WhatsApp: irá executar a aplicação do WhatsApp existente em seu sistema operacional, caso não seja localizado, será executado no navegador
  • Telefone
  • Com mensagem
  • Caixa de mensagem
  • Botão Enviar
  • Botão Gerar QR Code
